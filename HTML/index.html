Minecraft plug-in

The Yuuki Minecraft plugin is a Spigot plugin that connects Yuuki to your server so you can use her while playing. She comes with a bunch of useful commands such as /trade or the multi-use command /yuuki. Constant updates make you never get tired of Yuuki.
